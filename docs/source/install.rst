Installation
===============================

Prerequisites
-------------
* `Link Python <https://www.python.org/>`_
* `Link Numpy <http://www.numpy.org/>`_ 
* `Link Scipy <https://www.scipy.org/>`_ 
* `Link matplotlib <https://matplotlib.org/>`_ 


Downloading mcu
------------------------

.. code-block:: bash

    $ git clone https://github.com/hungpham2017/mcu

Python environment
------------------

.. code-block:: bash

    $ export PYTHONPATH=<path-to-mcu-package>:$PYTHONPATH
